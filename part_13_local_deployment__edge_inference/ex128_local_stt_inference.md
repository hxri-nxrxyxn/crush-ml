# ex128 & ex129: Advanced Local Inference (STT and LLMs)

### Speech-to-Text (ex128)
Local inference models like **OpenAI Whisper** allow you to transcribe audio without sending data to the cloud, ensuring privacy and saving costs.

### GGUF and llama.cpp (ex129)
GGUF is a binary format for Large Language Models (LLMs) that is optimized for CPU inference using 4-bit or 8-bit quantization. It allows you to run models like Llama-3 or Mistral on a standard laptop.
