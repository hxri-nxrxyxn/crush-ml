# ex130: The API Wrapper (FastAPI)

### Concept
The final step in deployment is making your model accessible via the web. **FastAPI** is the modern standard for building Python ML APIs.

### Why FastAPI?
- **Asynchronous**: Can handle many concurrent requests.
- **Type Safety**: Automatically validates the data sent by the user using Pydantic.
- **Documentation**: Automatically generates a Swagger UI (`/docs`) for testing your API.
