# ex080: TensorBoard

### Concept
TensorBoard is an amazing visualization tool provided with TensorFlow.

### Features
- **Scalars**: Track loss and accuracy in real-time.
- **Graphs**: Visualize the actual structure of your neural network.
- **Histograms**: See how weights change over time.
- **Images**: View the samples being processed.

### Usage
In Keras, simply add the `TensorBoard` callback and point it to a log directory.
