# ex060: XGBoost

### Concept
Extreme Gradient Boosting. This is an optimized implementation of Gradient Boosting that is famous for winning many Machine Learning competitions (Kaggle).

### Advantages
- **Speed**: Optimized for parallel processing.
- **Performance**: Includes advanced regularization (L1/L2) and handling of missing values.
- **Flexibility**: Can be used for classification, regression, and ranking.

### Study Note
XGBoost is often the "go-to" algorithm for structured/tabular data before moving to Deep Learning.
