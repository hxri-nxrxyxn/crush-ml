# ex001: Fetching and Downloading Data
### The Big Picture
Welcome to your first step in becoming a Machine Learning expert! In this project, we are working for the "Machine Learning Housing Corporation." Our goal is to build a model that predicts the **median housing price** in California districts. This is a **Supervised Learning** task because we have labels, and a **Regression** task because we predict continuous values.
### Why Automate?
Manual downloads are for hobbyists. Professionals automate data fetching for:
1. **Reproducibility**: Colleagues can run your code without hunting for links.
2. **Maintenance**: Automation handles data updates seamlessly.
3. **Setup**: Cloud servers and new environments set up with one command.
