# ex009: Scikit-Learn Imputer
### Scikit-Learn Design
1. **Estimators**: Objects that learn (`fit()`).
2. **Transformers**: Objects that change data (`transform()`).
3. **Predictors**: Objects that guess (`predict()`).
`SimpleImputer` is a Transformer that automates the "Fill with Median" strategy across all columns, making your code future-proof.
