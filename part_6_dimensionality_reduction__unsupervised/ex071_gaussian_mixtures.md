# ex071: Gaussian Mixture Models (GMM)

### Concept
A probabilistic model that assumes all the data points are generated from a mixture of a finite number of Gaussian distributions with unknown parameters.

### Comparison
- **K-Means**: Hard assignment (an instance belongs to exactly one cluster).
- **GMM**: Soft assignment (an instance has a probability of belonging to each cluster). This is called **Density Estimation**.

### Usage
GMMs are great for complex datasets where clusters have different sizes and elliptical shapes.
