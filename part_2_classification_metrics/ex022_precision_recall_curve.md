# ex022: Precision-Recall (PR) Curve

### Concept
A plot of precision versus recall for every possible threshold.

### When to use it
The PR curve is preferred when the **positive class is rare** (e.g., detecting cancer) or when you care more about false positives than false negatives.

### Analysis
You want the curve to be as close to the top-right corner as possible. The point where precision starts to fall sharply is usually the best threshold to pick.
