# ex038: Logistic Regression

### Concept
Used to estimate the probability that an instance belongs to a particular class. If the probability is $>50\%$, the model predicts the positive class.

### The Sigmoid Function
$$\sigma(t) = \frac{1}{1 + e^{-t}}$$
The output is always between 0 and 1.

### Decision Boundary
The line (or hyperplane) where the probability is exactly 50%. This is the boundary the model uses to separate classes.
